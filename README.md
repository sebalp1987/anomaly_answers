# 5Finder: Outlier Detection in the underwriting process

## Got a question?

Please, write to sebastian.mpalacio@gmail.com

## What is 5-Finder?

5Finder is an outlier detection model for Python created in Zurich Insurance Lta - Spain and as part of a research
project.

## Requerimentes

You need Python 3.6 or later to run 5Finder. See the requeriments.txt file.

## Quick Start

If Python on your system is configured properly you can run the program like this:

```
$ python main.py
```
